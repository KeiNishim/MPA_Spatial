# Spatial analysis of kidney from AAV patients and Healthy donors

## Overview
This repository provides code used for xenium analyses of kidney biopsies from 8 AAV (ANCA-associated vasculitis) patients and 4 healthy donors.

Nishide M, Nishimura K, et al, "Integrated single-cell spatial omics reveals neutrophil–epithelial crosstalk initiating fibrogenic remodeling in autoimmune glomerulonephritis", submitted. 

## Software requirements
・R version 4.4.1
・Seurat (v5.3.0)
・Monocle3 (v1.3.7)
・CellChat (v2.1.2)
・dbscan (v1.2.2)
・nichenetr (v1.1.1)
